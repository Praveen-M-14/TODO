# To-Do List Application

## Description
A simple To-Do List web application built using Python and Flask. The application allows users to create, read, update, and delete tasks, offering basic CRUD functionality.

## Features
- Create new tasks
- Read and view existing tasks
- Update tasks
- Delete tasks
